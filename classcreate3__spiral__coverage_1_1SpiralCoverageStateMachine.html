Detailed Description

State machine for implementing the spiral coverage algorithm.

This state machine coordinates a series of robot behaviors to implement an efficient coverage algorithm that combines spiral movement patterns with obstacle avoidance and navigation behaviors. It manages transitions between different behaviors based on sensor inputs and coverage goals.
