Detailed Description

Main ROS2 node for the Create3 straight line coverage algorithm.

This node provides an action server for the coverage action that implements a straight-line movement pattern with obstacle avoidance behaviors for area coverage. It manages robot navigation, docking, undocking, and handles various sensor inputs like hazard detection and odometry.
